# Typing Game - Sushi-da Style

A Flutter-based typing game inspired by the Sushi-da typing game.

## Features

- 🎮 Interactive typing game with multiple difficulty levels
- 📱 Bluetooth keyboard support
- 🎯 Real-time word matching
- 📊 Local high score system
- 😊 Character reactions based on combo count
- 🎨 Beautiful UI with Material Design 3

## Getting Started

### Prerequisites

- Flutter SDK >= 3.0.0
- Dart SDK >= 3.0.0
- Android Studio / Xcode (for emulator)

### Installation

```bash
# Clone repository
git clone https://github.com/your-username/typing-app.git
cd typing-app

# Install dependencies
flutter pub get

# Run the app
flutter run
